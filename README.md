# Pathya

## Overview

Pathya is an AI-powered nutrition analysis platform that helps users understand nutrient intake, identify deficiencies, and improve dietary habits.

## Goals

- Track nutrient consumption
- Compare intake against recommended daily values
- Identify deficiencies and excesses
- Provide AI-assisted food parsing
- Deliver actionable nutrition insights

## Tech Stack

### Frontend
- Angular
- Angular Material
- SCSS

### Backend
- ASP.NET Core Web API
- Entity Framework Core

### Database
- PostgreSQL

### Hosting
